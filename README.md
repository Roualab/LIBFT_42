## Description

**libft** is a custom C library created from scratch as part of the 42 curriculum.

The goal of this project is to:
- Understand how common C standard library functions work
- Re-implement them without using the original ones
- Build a reusable library (libft.a) that can be used in future projects

This project helps learn:
- Memory management
- String manipulation
- Linked lists
- Clean and structured C code

---

## Instructions

This project uses Make command to compile and manage the library easily.

****** make
Compiles all .c files and creates the static library libft.a.

****** make all
Same as make. It builds the library from scratch if needed.

****** make clean
Removes all temporary object files (.o).
This keeps the folder clean without deleting the library.

****** make fclean
Removes everything generated by compilation, including libft.a.

****** make re
Fully rebuilds the project by running fclean followed by make.

These commands help you compile, clean, and rebuild the project quickly and safely.

##  Library Content (Function List)

### Character checks

* **ft_isalpha** – Checks if a character is a letter.
* **ft_isdigit** – Checks if a character is a digit.
* **ft_isalnum** – Checks if a character is a letter or a digit.
* **ft_isascii** – Checks if a character is a valid ASCII character.
* **ft_isprint** – Checks if a character is printable.
* **ft_toupper** – Converts a lowercase letter to uppercase.
* **ft_tolower** – Converts an uppercase letter to lowercase.

---

### String functions

* **ft_strlen** – Returns the length of a string.
* **ft_strlcpy** – Copies a string safely with size limit.
* **ft_strlcat** – Appends a string safely with size limit.
* **ft_strchr** – Finds the first occurrence of a character in a string.
* **ft_strrchr** – Finds the last occurrence of a character in a string.
* **ft_strncmp** – Compares two strings up to a given length.
* **ft_strnstr** – Finds a substring inside another string.
* **ft_strdup** – Duplicates a string using allocated memory.
* **ft_substr** – Extracts a substring from a string.
* **ft_strjoin** – Joins two strings into a new one.
* **ft_strtrim** – Removes characters from the start and end of a string.
* **ft_split** – Splits a string using a delimiter.
* **ft_strmapi** – Creates a new string by applying a function to each character.
* **ft_striteri** – Applies a function to each character of a string (in place).

---

### Memory functions

* **ft_memset** – Fills memory with a byte value.
* **ft_bzero** – Sets memory to zero.
* **ft_memcpy** – Copies memory (no overlap).
* **ft_memmove** – Copies memory safely (handles overlap).
* **ft_memchr** – Finds a byte in memory.
* **ft_memcmp** – Compares two memory areas.
* **ft_calloc** – Allocates and zeroes memory.

---

### Conversions

* **ft_atoi** – Converts a string to an integer.
* **ft_itoa** – Converts an integer to a string.

---

### File descriptor output

* **ft_putchar_fd** – Writes a character to a file descriptor.
* **ft_putstr_fd** – Writes a string to a file descriptor.
* **ft_putendl_fd** – Writes a string followed by a newline.
* **ft_putnbr_fd** – Writes an integer to a file descriptor.

---

### Linked list

* **ft_lstnew** – Creates a new list node.
* **ft_lstadd_front** – Adds a node at the beginning of a list.
* **ft_lstadd_back** – Adds a node at the end of a list.
* **ft_lstsize** – Counts the number of nodes in a list.
* **ft_lstlast** – Returns the last node of a list.
* **ft_lstdelone** – Deletes a single node.
* **ft_lstclear** – Deletes all nodes of a list.
* **ft_lstiter** – Applies a function to each node’s content.
* **ft_lstmap** – Creates a new list by transforming each node.
